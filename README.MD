MOVIE-APP-SERVER

TECH STACK :

- Express
- NodeJs
- Sequelize
- Javascript
- JsonWebToken
- Bcrypt
- Postgresql
- Dotenv
- axios

3rd Party API

- The movie db
  login on https://www.themoviedb.org/ to get MOVIE_KEY

To run this server

- npm i
- npx sequelize db:create
- npx sequelize db:migrate
- npx sequelize db:seed:all
- npm run start
