MOVIE-APP-SERVER

TECH STACK :

- Express
- NodeJs
- Sequelize
- Javascript
- JsonWebToken
- Bcrypt
- Postgresql
- Dotenv

To run this server

- npm i
- npx sequelize db:create
- npx sequelize db:migrate
- npx sequelize db:seed:all
- npm run start
